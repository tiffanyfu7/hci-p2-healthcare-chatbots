To run patient chatbot
```
cd patient
pip intall -r requirements.txt
streamlit run patient_chatbot.py
```
To run caregiver chatbot
```
cd caregiver
pip intall -r requirements.txt
streamlit run src/Home.py
```
To run doctor chatbot
```
cd doctor
pip intall -r requirements.txt
streamlit run doctor_notes.py
```
